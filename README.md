# Python-EDA-SkySport
# Python Data Analysis: EDA, Retention и Сегментация (SkySport)

## Описание проекта
В данном репозитории представлены Jupyter Notebooks с анализом клиентской базы фитнес-приложения. Проект охватывает полный цикл работы с данными: от очистки и поиска аномалий до когортного анализа, расчета оттока (Churn) и выявления профиля самых ценных клиентов (Power Users).

## Стек технологий
* **Язык:** Python
* **Библиотеки:** Pandas, NumPy, Matplotlib, Seaborn

## Структура проекта
* `skysport_eda_and_cleaning.ipynb` — Разведочный анализ (EDA), обработка пропусков и удаление статистических выбросов (IQR), базовая демография.
* `skysport_retention_and_ltv.ipynb` — Расчет продуктовых метрик. Выделение Premium-сегмента, профилирование лояльных пользователей (Power Users), расчет динамики клиентской базы по месяцам (New, Active, Gone).

## Ключевые бизнес-результаты
1. Выявлен портрет Premium-клиента (индивидуальные тренировки) с расчетом среднего чека.
2. Проведена кластеризация аудитории на группы активности для таргет-маркетинга.
3. Построена визуализация динамики оттока и притока новых клиентов (Retention & Churn rate) по месяцам.
